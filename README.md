## 1 安装 tinyxml 

把tinyxml的makefile文件中105行，就是Output注释下的LD改为AR指定生成静态库就可以了

或

${OUTPUT}: ${OBJS}
${LD} -o $@ ${LDFLAGS} ${OBJS} ${LIBS} ${EXTRA_LIBS}
替换成
${OUTPUT}: ${OBJS}
	${AR} $@ ${OBJS}
	${RANLIB} $@
应该就可以了
